# 常见问题 (FAQ)

本页面汇总了使用 ShadowBoard 过程中的常见问题与解决方案。

---

## ❓ 故障排除

### Q: 浏览器窗口打开了，但一直提示"找不到输入框"？
**A**:
1.  **手动验证**：请检查弹出的浏览器窗口是否已加载完成。
2.  **登录状态**：如果页面跳转到了登录页，请在窗口内手动完成登录。
3.  **选择器失效**：如果平台更新了 UI，请在 `src/core/config.py` 中更新相应的 CSS 选择器。

### Q: 性能自测 (perf_check) 失败怎么办？
**A**: `perf_check` 的默认阈值是针对高性能开发机器的。在资源受限的环境（如云服务器）下，可以通过设置环境变量 `CI=true` 来自动放宽 5 倍的限制。

### Q: 如何完全重置系统状态？
**A**:
```powershell
# 删除所有本地缓存和数据库
Remove-Item -Path .semi_agent/ -Recurse -Force
```
