# 网页 AI 半自动助手

本工具用于把对话需求转为网页 AI 的半自动执行流程

## 核心能力
- 多平台支持
  DeepSeek 网页
  Kimi 网页
  通义千问 网页
  豆包 网页
  智谱清言 网页
  文心一言 网页
- 平台预设
  一键应用网址与发送方式
- 新手引导
  进度状态 下一步建议 常见问题
- 执行链路
  打开登录浏览器 登录检查 冒烟测试 任务执行
- 结果管理
  导出文本 导出 Markdown
- 诊断能力
  历史筛选 健康检查 错误日志
- 帮助文档
  页面内可刷新接口文档并导出

## 快速开始
1 运行 run.bat
2 在 平台与参数 页选择平台并应用预设
3 保存参数
4 点击 打开登录浏览器 并完成登录
5 点击 登录完成检查
6 勾选冒烟测试确认并执行冒烟测试
7 进入 执行任务 开始正式任务

## 严格验证结果
- 事件自动化测试通过
- 静态检查通过
- 编译检查通过

## 文件说明
- web_app.py 页面与事件逻辑
- main.py 浏览器自动化核心
- tests/test_web_app_events.py 事件自动化测试
- DELIVERY_CHECKLIST.md 交付清单与完成状态
